+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

					Как использовать сниппеты
					-------------------------
					
1. Подключить файл стилей сниппета (LINK) или вставить код сниппета в файл стилей проекта
2. В HTML использовать классы сниппета

+++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

					
-----------
	Тени
-----------
shadow.scc
-----------
1. Простая тень - приподнятый блок				class="raised-shadow"
2. Приподнятые углы								class="drop-shadow lifted"
3. Искивленные углы								class="drop-shadow curled"
4. Перспектива									class="drop-shadow perspective"
5. Искривление	- с двух сторон вертикаль		class="drop-shadow curved"
				- слева							class="drop-shadow curved vert-left"
				- справа						class="drop-shadow curved vert-right"
				- с двух сторон горизонталь 	class="drop-shadow curved hor">
				- сверху						class="drop-shadow curved hor-top">
				- снизу							class="drop-shadow curved hor-bottom">
6. Повернутый блок (с приподнятыми углами)		class="drop-shadow lifted rotated"
